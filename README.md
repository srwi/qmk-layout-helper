# QMK Layout Helper

Desktop overlay for learning new QMK keyboard layouts.

## Requirements

- [QMK CLI](https://docs.qmk.fm/cli)
- Keyboard information json file, e.g. `qmk info -kb planck/rev6 -m -f json > keyboard_info.json`

## Usage

Run `qmk-layout-helper keyboard_info.json [-l <LAYOUT NAME>]`
